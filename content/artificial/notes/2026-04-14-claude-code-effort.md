---
title: Claude Code effort is a real lever
date: 2026-04-14
tags: [claude-code, prompting, workflow]
---

Claude Code's effort setting changes how much thinking budget it spends on the task. This is not cosmetic. If the problem needs planning, tradeoff analysis, or careful code changes, more effort often matters more than rewriting the prompt ten different ways.

The easy trigger is language. Phrases like "think hard" and especially `ultrathink` push it to spend more effort. If the task is planning-first, `ultraplan` is useful because it biases toward a more deliberate plan before action.
