I added colored tabs for CC and OC in `~/.zshrc` with `tab-cc` for claude code and `tab-oc` for opencode. This should have been obvious earlier. Need to add one for Codex as well. Since i starting using that too recently.

When you're bouncing between agents, visual separation matters. It removes a tiny bit of friction and a tiny bit of risk every single time you switch context.
